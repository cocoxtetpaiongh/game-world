## Overview

This repository stores the foundational documents for a large-scale mythology RPG project.

The current direction is an open-world action RPG inspired by Southeast Asian mythology, ancient celestial technology, reincarnation, and time-fractured history.

The project begins with documentation first:

1. Story foundation
2. World bible
3. Character bible
4. Gameplay design
5. Art direction
6. Technology/tool selection
7. Market strategy
8. Prototype roadmap

The current goal is not to build the full game immediately. The immediate goal is to define a clear creative foundation that can later guide development, asset creation, tools, production planning, and marketing.

---

## Current Core Idea

**Age of Legends: Echoes of Heaven** is an open-world action RPG set in a mythological Southeast Asia where dragons, nagas, spirits, celestial ruins, reincarnation, and time-fractured ancient technology collide.

The game begins as fantasy.

Over time, the player discovers that the so-called gods were an advanced civilization whose technology became myth.

The player is the reincarnation of the final Celestial Emperor, the ruler who created the Heaven Engine, a machine designed to protect civilization by controlling history.

The central question is:

> Should this world continue to exist if it was born from a broken timeline?

## Development Philosophy

The project follows this order:

```text
Story → World → Characters → Gameplay → Art Direction → Technology → Marketing
```

The purpose is to avoid choosing tools too early before the game identity is clear.
